# RESEARCH.md — Исследование проекта Quiz Electro

*Исследование проведено с помощью 5 параллельных агентов. Данные основаны на знаниях до августа 2025 года.*

---

## 1. Конкуренты

### 1.1 Marquiz (marquiz.ru)

**Тип:** SaaS-конструктор квизов для лидогенерации

**Функции:** визуальный редактор, ветвление вопросов, прогресс-бар, калькулятор стоимости, интеграции с AmoCRM / Bitrix24 / Telegram, аналитика по шагам, встраивание на сайт.

**Хорошо:**
- Калькулятор цены + прогресс-бар = главный конверсионный тандем
- Ветвление: разные вопросы для разных сценариев (замена проводки vs. установка розеток)
- Мобильная адаптация из коробки
- Telegram-интеграция — критично для СНГ

**Плохо:**
- Бесплатный тариф: 1 квиз, 3 ответа/мес — бесполезен для реальной работы
- Платный от ~990 руб. РФ (~30–35 BYN) — барьер оплаты из Беларуси
- Брендинг Marquiz на бесплатном тарифе
- Аналитика базовая, нет записи сессий

**Для белорусского электрика:** отличный эталон UX, но платёжный барьер делает его недоступным. Самописный квиз выгоднее: разово разработать и платить $3–5/мес за хостинг.

---

### 1.2 Envybox (envybox.io)

**Тип:** SaaS-платформа конверсионных инструментов (квиз — один из модулей)

**Функции:** квиз + обратный звонок + онлайн-чат, шаблоны для ремонта/строительства, exit-intent (поп-ап при попытке закрыть вкладку), A/B тестирование, автодозвон оператора.

**Хорошо:**
- Exit-intent квиз — перехватывает уходящего пользователя
- Шаблон «Рассчитайте стоимость ремонта» напрямую применим к электрике
- Автодозвон повышает конверсию больше, чем обычная форма

**Плохо:**
- Платишь за всю экосистему, даже если нужен только квиз
- Цена от ~2000 руб. РФ/мес — избыточно для одного мастера
- Та же проблема с оплатой из Беларуси

**Для белорусского электрика:** полезен как источник идей по UX-триггерам, практически неприменим из-за цены.

---

### 1.3 Typeform (typeform.com)

**Тип:** Международный SaaS-конструктор форм и квизов

**Функции:** формат «один вопрос на экране», условная логика, разнообразные типы полей, интеграции через Zapier / Google Sheets / HubSpot, API.

**Хорошо:**
- Лучший UX среди конкурентов: анимации, минималистичный дизайн, фокус на одном вопросе
- Один вопрос на экране снижает когнитивную нагрузку — повышает completion rate
- Работает в любой стране

**Плохо:**
- Платный тариф от $25/мес — дорого для частного мастера
- Интерфейс только на английском
- Telegram-уведомления только через сторонние сервисы
- Бесплатный лимит 10 ответов/мес — исчерпывается за неделю при рекламе

**Для белорусского электрика:** главная идея для заимствования — «один вопрос на экране». По цене неприменим.

---

### 1.4 LeadQuizzes (leadquizzes.com)

**Тип:** Международный SaaS, специализируется на квизах для лидогенерации

**Функции:** score-based логика (ответы суммируются → «диагноз»), шаблоны для home services / contractors, аналитика drop-off по шагам, A/B тестирование.

**Хорошо:**
- Score-based: ответы дают «диагноз» электрики клиента — мощный конверсионный крючок
- Шаблоны под contractors — минимум адаптации
- Аналитика показывает, на каком вопросе люди уходят

**Плохо:**
- Цена от $49/мес
- Нет Telegram-интеграции
- Только английский интерфейс

**Для белорусского электрика:** ценна концепция «диагностики» (а не просто сбора контактов). Реализовать самостоятельно.

---

### 1.5 Interact (tryinteract.com)

**Тип:** Международный SaaS, фокус на виральности и персонализации

**Функции:** три формата (personality / scored / assessment), lead gate до или после результата, условная логика, интеграции с email-сервисами.

**Хорошо:**
- Assessment-формат («оцените состояние вашей электрики») — точное попадание в нишу
- Lead gate «до результата» — пользователь вложился в квиз и охотнее оставляет контакт
- Шаблоны с понятной структурой 5–7 вопросов

**Плохо:**
- Платный от $39/мес
- Нет Telegram-интеграции, нет русского

**Для белорусского электрика:** идея lead gate (форма перед показом результата) стоит взять в самописный квиз.

### Конкурентное преимущество самописного квиза

Все платные платформы стоят $25–50/мес и имеют платёжные барьеры для Беларуси. Самописный квиз: разовая разработка, хостинг ~$3–5/мес, полный контроль над дизайном, Telegram-интеграция без посредников, никакого чужого брендинга.

### Что взять из лучших практик

- **Один вопрос на экране** (Typeform) — снижает нагрузку, повышает completion rate
- **Прогресс-бар с числами** (Marquiz) — «осталось 2 вопроса» удерживает
- **Score-based диагностика** (LeadQuizzes) — ответы дают «диагноз», а не просто сбор данных
- **Lead gate перед результатом** (Interact) — форма после вложения в квиз
- **Персонализированный результат** — не «мы свяжемся», а конкретный вывод с диапазоном цен
- **Telegram-уведомление** (Marquiz, Envybox) — мастер получает заявку мгновенно
- **Не брать:** виральные механики, таймеры-пустышки, калькулятор «точной» стоимости (электрика слишком вариативна)

## 3. UX и сценарии

### 3.1 Три типичных сценария

#### Сценарий А: Быстрый заказ («Я знаю, чего хочу»)

**Профиль:** принял решение, ищет цену или конкретного мастера. Пришёл по рекламе.

**Поведение:**
- Сканирует квиз по диагонали, не читает пояснения
- Хочет дойти до результата за 2–3 минуты
- Бросает квиз, если не видит движения к цене/результату
- Заполняет только обязательные поля

**Критичные точки трения:** длина квиза (> 5–6 шагов), отсутствие прогресс-бара, нерелевантные вопросы, обязательный телефон до результата.

**Что удерживает:** первый вопрос точно совпадает с его задачей, быстрый переход к результату с диапазоном цен.

---

#### Сценарий Б: Первое знакомство («Не знаю, с чего начать»)

**Профиль:** столкнулся с проблемой впервые, не знает терминологии, боится показаться некомпетентным.

**Поведение:**
- Читает все варианты ответов, возвращается назад
- Выбирает «средний» вариант при неуверенности
- Ценит пояснения рядом с вариантами («что это значит?»)
- Охотно оставляет контакт в обмен на экспертный совет

**Критичные точки трения:** профессиональный жаргон без объяснений («стяжка», «дифавтомат»), только текстовые варианты без иллюстраций, ощущение допроса, а не помощи.

**Что удерживает:** нейтральный дружелюбный тон, пояснения к терминам, формулировка результата как «рекомендация эксперта».

---

#### Сценарий В: Сомневающийся («Сравниваю мастеров»)

**Профиль:** уже смотрел 2–3 конкурента, получил одно КП. Чувствителен к цене и доверию.

**Поведение:**
- Заполняет квиз до конца, но не оставляет телефон, если результат — «мы перезвоним»
- Готов оставить email (менее обязывает), но не телефон на старте
- Читает отзывы рядом с формой контактов

**Критичные точки трения:** результат без цифр, агрессивный поп-ап, принудительный телефон до результата.

**Что удерживает:** диапазон цен в результате, блок с 2–3 отзывами рядом с формой, конкретный следующий шаг («Мастер позвонит в течение часа»).

---

### 3.2 Лучшие UX-практики

**Структура:**
- Один вопрос на экране — снижает когнитивную нагрузку (принцип Progressive Disclosure, Nielsen Norman Group)
- Прогресс-бар с числами: «Вопрос 3 из 6» — completion rate выше на 20–28%
- Кнопка «Назад» всегда доступна — блокировка вызывает сильное раздражение
- Оптимум: 5–7 вопросов. После 8 — резкий рост отказов

**Контент:**
- Первый вопрос — самый лёгкий и конкретный (foot-in-the-door: первый клик = психологическое вовлечение)
- Микрокопии под вариантами снимают неуверенность новичков
- Нейтральный тон: «Что вам важно?» работает лучше, чем «Хотите сэкономить?»
- Визуальные карточки с иконками вместо текста: +10–20% к completion rate (данные Typeform 2023)

**Форма контактов:**
- Контакты — после показа ценности, не до
- Минимум полей: имя + телефон. Каждое доп. поле снижает конверсию на 10–15%
- «Получить расчёт» конвертирует лучше, чем «Отправить заявку»
- «Без обязательств» снижает психологический барьер

---

### 3.3 Антипаттерны — что бесит пользователей

| Антипаттерн | Почему раздражает |
|---|---|
| > 7–8 вопросов без результата | Ощущение «меня используют», completion rate падает |
| Нет прогресс-бара | «Не знаю, сколько ещё» — лучше уйду |
| Нет кнопки «Назад» | Ошибся — начинать заново? Закрываю |
| Телефон на шаге 1–2 | Воспринимается как спам-ловушка |
| Результат «мы перезвоним» | После 6 вопросов — ощущение потраченного времени впустую |
| Профессиональный жаргон | «Что такое дифавтомат?» — пользователь не электрик |
| Фейковый таймер | Клиенты знают, что он фейковый. Обесценивает оффер |
| Автопереход без подтверждения | Ощущение потери контроля |

### 3.4 Квизы и конверсия: данные

| Метрика | Стандартный лендинг | Квиз-воронка |
|---|---|---|
| Средний CR в лид | 1–4% | 8–25% |
| Стоимость лида | базовая | на 30–60% ниже |
| Качество лида | среднее | выше (квалифицирован через ответы) |
| Время на странице | 30–90 сек | 2–5 минут |

**Кейсы (открытые источники, 2022–2024):**
- Ремонт квартир (кейс агентства Completo, 2023): CR лендинга 2,1% → CR квиза 11,4%, CPL снизился с 3200 до 1100 руб. РФ
- Установка окон (кейс Marquiz): CR квиза 18% vs. лендинг 3,8%
- Клининг (A/B тест Typeform): квиз дал +62% к конверсии в заявку
- По данным Outgrow: ниши «home services» (ремонт, сантехника, электрика) — CR квиза 15–22%

**Факторы, повышающие CR квиза:**
- Показ диапазона цен до запроса контакта: +15–30%
- Визуальные ответы вместо текста: +10–20%
- Прогресс-бар с числами: +8–12%
- Социальное доказательство рядом с формой: +10–15%

**Факторы, снижающие CR:**
- > 7 вопросов: −20–40% completion
- Обязательный телефон на старте: −50–70%
- Результат без цифр: −25–35%
- Плохая мобильная адаптация: −30–50%

### 4.2 Отправка заявок (без бэкенда)

| Критерий | Formspree | Telegram Bot API | EmailJS |
|---|---|---|---|
| Бесплатный лимит | 50/мес | Без лимитов | 200 писем/мес |
| Куда приходит | Email | Telegram-чат | Email |
| Сложность настройки | Минимальная | Средняя (~30 строк JS) | Низкая |
| Попадание в спам | Возможно | Никогда | Возможно |
| Платный тариф | $10/мес (1000/мес) | $0 | $9/мес (1000/мес) |

**Схема Telegram Bot API без бэкенда:**
```javascript
// Прямой fetch из JS (токен в коде — допустимо для одностороннего бота)
fetch(`https://api.telegram.org/bot${TOKEN}/sendMessage`, {
  method: 'POST',
  body: JSON.stringify({ chat_id: CHAT_ID, text: message })
})
```

**Рекомендация: Telegram Bot API** — бесплатно, без лимитов, заявки приходят мгновенно туда, где мастер уже есть. 50 заявок/мес Formspree исчерпается за неделю активной рекламы.

## 5. Монетизация

### 5.1 Шесть моделей монетизации

#### 1. Наценка на материалы ★★★★★

**Механика:** закупка по дилерской цене у ABB/Schneider/Legrand, продажа клиенту по розничной. Дельта — 10–20%.

**Цифры:** на квартире 70 м² материалы ~1800 BYN. Наценка 12% = +216 BYN к доходу. На полной замене проводки (материалы 2500 BYN) — наценка 300–500 BYN.

**Что работает:**
- Расходники (гофра, стяжки, дюбели) — наценка 30–50%, никто не проверяет
- Розетки/выключатели premium-серий — клиент выбирает эмоционально, мастер рекомендует
- Автоматы и кабель — клиент часто проверяет цену, лучше брать за «под ключ комплектацию»

**Порог входа:** минимальный — встроено в основной процесс работы.

---

#### 2. Реферальная сеть мастеров ★★★★☆

**Принцип:** каждый клиент электрика — потенциальный клиент сантехника, плиточника, маляра.

```
Электрик ←→ Сантехник
         ←→ Плиточник / отделочник
         ←→ Установщик кондиционеров
         ←→ Монтажник видеонаблюдения
         ←→ Дизайнер интерьера
```

**Форматы:** устный обмен лидами / фикс 20–50 BYN за закрытый заказ / 5–10% от чека.

**Цифры:** сантехник даёт 3–4 тёплых лида в месяц по 5% с чека 600 BYN = 90–120 BYN пассивного дохода без усилий.

**Кейс (публичные чаты BY, ~2023):** мастер собрал сеть из 4 смежников → 8–12 входящих реферальных заявок в месяц без рекламы → ~800–1200 BYN доп. дохода.

---

#### 3. Продажа / передача лидов ★★★★☆

**Когда актуально:** мастер перегружен, заказ не по специализации, или далеко географически.

| Формат | Цена |
|---|---|
| Горячий лид другому электрику | 30–100 BYN flat |
| Процент от выполненного заказа | 5–10% |

**Цифры:** 5 заявок/нед, закрывает 3, передаёт 2 по 50 BYN = +100 BYN/нед = ~400 BYN/мес без доп. работы.

**Важно:** продавать только лиды, где клиент дал согласие на передачу данных.

---

#### 4. Абонентское обслуживание ★★★☆☆

| Пакет | Что входит | Цена/год |
|---|---|---|
| Базовый | 1 визит, диагностика щитка | 80–120 BYN |
| Стандарт | 2 визита, замер изоляции, приоритетный выезд | 180–250 BYN |
| Премиум | Ежеквартальный контроль, аварийный выезд 24/7 | 400–600 BYN |

**Целевые клиенты:** владельцы частных домов со старой проводкой, малый бизнес (кафе, магазины — у них есть журнал ТО).

**Ключ к продаже:** предлагать абонемент сразу после завершения объекта как «продолжение гарантии».

**Реалистично:** 10 договоров × 200 BYN = 2000 BYN/год стабильной базы. Для частных клиентов BY продать сложно — культуры профилактики нет. Для коммерции — работает.

---

#### 5. Партнёрские программы с магазинами ★★☆☆☆

**Форматы в BY:** карта мастера с откатом 3–7%; дилерский статус ABB/Schneider (скидка 10–15% = маржа); промокоды на electro.by, 220.by (2–5%).

**Ограничения:** требует оборотного капитала и логистики. Для одиночки — сложнее, чем наценка на материалы при заказе.

---

#### 6. Обучение и консультации онлайн ★★☆☆☆

| Формат | Цена | Трудозатраты |
|---|---|---|
| Онлайн-консультация 1:1 (Zoom/Telegram) | 20–50 BYN/час | Низкие |
| Разбор схемы щитка | 30–80 BYN | Средние |
| Мини-курс «Сам себе электрик» (5–8 видео) | 49–149 BYN | Высокие один раз |
| Telegram-группа с подпиской | 9–19 BYN/мес | Средние (контент) |

**Реалистично для BY:** консультации 1:1 работают как апсейл к основному сервису. Курсы — только с медиаприсутствием (YouTube, Reels). Без аудитории — нет продаж.

### 5.3 Квиз как точка входа для нескольких потоков

```
                    КВИЗ
                      │
          ┌───────────┼───────────┐
          ▼           ▼           ▼
    [Тип объекта] [Бюджет]  [Срочность]
          │           │           │
     Основной     Продажа      Повышенный
      заказ         лида          тариф
   (работа +     другому        (×1.5–2)
   материалы)    мастеру
          │
          ▼
   «Нужен ли сантехник/плиточник?»
          │
          ▼
   Реферальная продажа смежнику
```

**Конкретные связки:**

- **Квиз → Сегментация → Апсейл материалов:** вопрос «Какие розетки предпочитаете?» → «Не знаю, посоветуйте» → мастер рекомендует premium-серию с наценкой
- **Квиз → Смежные потребности → Реферал:** вопрос «Планируете другие работы?» → «Да, сантехника» → передаёт контакт партнёру за 30–50 BYN
- **Квиз → Оценка объекта → Абонемент:** «Когда последний раз проверяли электрику?» → «Более 5 лет» → в конце предложение абонемента 99 BYN/год
- **Квиз → «Холодный» лид → Чек-лист → Конверсия позже:** не готов заказать → получает PDF «5 признаков опасной электрики» → конверсия через 1–3 месяца

### 5.4 Карта приоритетов для белорусского рынка

```
БЫСТРЫЙ СТАРТ (0–3 месяца):
├── Наценка на материалы          → +15–20% к выручке
├── Реферальная сеть мастеров     → 100–300 BYN/мес
└── Продажа невостребованных лидов → 100–400 BYN/мес

СРЕДНИЙ ГОРИЗОНТ (3–12 месяцев):
├── Абонемент для коммерции       → 3–10 договоров = 400–2000 BYN/мес
└── Партнёрки с поставщиками      → дилерский статус ABB/Schneider

ДОЛГИЙ ГОРИЗОНТ (12+ месяцев):
└── Обучение онлайн               → только с медиаприсутствием
```

**Главный принцип:** квиз — не просто форма заявки. Это инструмент сбора структурированных данных о клиенте, который позволяет одному мастеру маршрутизировать клиентов по 3–4 потокам дохода одновременно.
